# Email_Classification
Use Llama 3.2 to try to classify incoming email messages into one of the following categories:
+ Update for cargo transportation
+ Advertisement
+ Spam/Phishing
+ Request for Quote
+ Offer
+ Accepted or Not 
+ Booking Note
+ Reference Number Execution
